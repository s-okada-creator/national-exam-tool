## 2. テストモードの確認

### 2.1 ページ読み込み
- [ ] ページが正常に読み込まれる
- [ ] 問題が表示される
- [ ] 選択肢が表示される
- [ ] ナビゲーションボタン（前へ/次へ）が表示される
- [ ] 「テスト終了」ボタンが表示される

### 2.2 テーマ・キーワードの非表示確認
- [ ] **重要**: 「テーマ・キーワード」のセクションが表示されていない
- [ ] 選択肢の上にテーマ・キーワードの情報がない

### 2.3 タイマーの動作確認
- [ ] タイマーが表示されている（右上）
- [ ] タイマーがカウントダウン形式で表示されている（例: `03:30` → `03:29` → `03:28`）
- [ ] タイマーが1秒ごとに減っていく
- [ ] タイマーの初期値が正しい（問題数 × 70秒、例: 3問なら210秒 = `03:30`）

### 2.4 タイマーの継続確認（重要）
- [ ] 1問目の選択肢をクリック
- [ ] 2問目に自動移動する
- [ ] **タイマーが継続してカウントダウンしている**（リセットされていない）
- [ ] タイマーの値が前の問題から継続している（例: 1問目で`03:25`なら、2問目でも`03:24`など）
- [ ] 2問目の選択肢をクリック
- [ ] 3問目に自動移動する
- [ ] **タイマーが継続してカウントダウンしている**

### 2.5 タイマーの警告色確認
- [ ] 残り180秒（`03:00`）以下でタイマーが黄色に変わる
- [ ] 残り60秒（`01:00`）以下でタイマーが赤色・太字に変わる

### 2.6 自動移動の確認
- [ ] 選択肢をクリックすると、**即座に**次の問題に自動移動する
- [ ] 遅延がない（1.5秒待たない）
- [ ] 最後の問題では自動移動しない

### 2.7 ナビゲーションボタンの確認
- [ ] 「前へ」ボタンで前の問題に戻れる
- [ ] 「次へ」ボタンで次の問題に進める
- [ ] タイマーが継続している（ナビゲーション後もリセットされない）
- [ ] プログレスバーが正しく更新される

### 2.8 テスト終了の確認
- [ ] 「テスト終了」ボタンが動作する
- [ ] 確認ダイアログが表示される

## 問題が発生した場合

1. **ブラウザのキャッシュをクリア**
   - Chromeの設定 → プライバシーとセキュリティ → 閲覧履歴データの削除
   - 「キャッシュされた画像とファイル」にチェックを入れて削除

2. **ハードリロード**
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

3. **開発者ツールのConsoleでエラーを確認**
   - エラーメッセージを記録
   - スクリーンショットを撮影

4. **Networkタブでリクエストを確認**
   - 失敗しているリクエストを確認
   - ステータスコードを確認
